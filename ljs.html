JavaScript Standards

Below is the complete list of topics that will be tested as part of the JavaScript sections of the written test and team challenge. Links to study these topics are also included.

If students have not yet learned about HTML and CSS, we recommend they start here instead. The list of resources below is intended for students that have already completed the Mozilla Development Network's HTML and CSS tutorials. We recommend it be used to check a student's understanding of the material and brush up on forgotten topics, not as a primary resource for learning.

Unlike the HTML and CSS sections, our JS standards only include material from the first two sections of MDN's JavaScript module. The other sections still include important content for aspiring web developers, but they are not necessary for this competition.

Remember, if you are having any trouble with the content, please don't hesitate to reach out to the 3C team for help here.
